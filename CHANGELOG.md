# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-13

### Added
- Multi-source RSS feed fetching (BBC, CNN, Reuters, AP, NPR)
- OpenAI-powered article summarization with configurable parameters
- AI importance ranking with robust response parsing
- Daily briefing generation in news-anchor style
- LLM auditor integration for budget tracking and quality scoring
- Certification report export (PDF/HTML/MD)
- Rich terminal CLI with panels, tables, and progress indicators
- Streamlit web dashboard with interactive controls
- Environment-based configuration via `.env`
- Unit tests with mocked OpenAI and feedparser calls
- CI/CD pipeline via GitHub Actions
